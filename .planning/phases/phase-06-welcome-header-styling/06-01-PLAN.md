# Plan: Welcome Header Styling

## Objective

Apply visual styling improvements to the chat panel welcome header to improve visual hierarchy and readability. The changes include making the headline bold, increasing subtitle size, and improving spacing between elements.

**Purpose**: Enhance the visual presentation of the chat panel header for better user experience
**Output**: Styled welcome header with bold headline, larger subtitle, and improved spacing

## Execution Context

**Phase**: 6 - Welcome Header Styling
**Plan**: 06-01
**Estimated Scope**: Trivial (~30 minutes)
**Type**: Styling / Visual polish

### Key Files

- `src/App.tsx` - Contains the chat panel header section (lines ~2546-2574)

### Current State

The chat panel header currently has:
```tsx
<div className="p-5 flex-shrink-0">
  <p className="text-xl font-medium text-text-primary mb-2">
    Welcome to console.
  </p>
  <p className="text-sm text-text-secondary mb-5">
    Console is an Agent-Driven UI. You give me a command, and I retrieve.
  </p>
  {/* Quick Action Buttons */}
</div>
```

### Requirements

1. **Bold headline**: Change "Welcome to Console" from `font-medium` to `font-bold`
2. **Larger subtitle**: Increase subtitle text from `text-sm` (14px) to `text-base` (16px)
3. **More padding**: Increase spacing between header elements (headline, subtitle, buttons)

## Context

The chat panel is located on the right side of the application, inside a floating card-style container with a glassmorphism effect (`bg-white/70 backdrop-blur-sm`). The header section is at the top of this panel and includes:

- Welcome headline
- Description subtitle
- Quick action buttons

## Tasks

### Task 1: Make Welcome Headline Bold

**Goal**: Increase visual weight of the "Welcome to Console" headline

**Files**: `src/App.tsx`

**Action**:
Change the headline's Tailwind class from `font-medium` to `font-bold`:

```tsx
// From:
<p className="text-xl font-medium text-text-primary mb-2">

// To:
<p className="text-xl font-bold text-text-primary mb-2">
```

**Verify**: Visual inspection - headline appears bolder than before

**Done**: "Welcome to console." text is bold (font-weight: 700)

---

### Task 2: Increase Subtitle Text Size

**Goal**: Make the subtitle text more readable by increasing its size

**Files**: `src/App.tsx`

**Action**:
Change the subtitle's Tailwind class from `text-sm` (14px) to `text-base` (16px):

```tsx
// From:
<p className="text-sm text-text-secondary mb-5">

// To:
<p className="text-base text-text-secondary mb-5">
```

**Verify**: Visual inspection - subtitle text is noticeably larger than before

**Done**: Subtitle "Console is an Agent-Driven UI..." uses 16px font size

---

### Task 3: Add More Padding Between Header Elements

**Goal**: Improve visual breathing room between headline, subtitle, and buttons

**Files**: `src/App.tsx`

**Action**:
Increase the margin-bottom values to create more separation:

1. Headline margin: Change `mb-2` (8px) to `mb-4` (16px)
2. Subtitle margin: Keep `mb-5` (20px) or optionally increase to `mb-6` (24px) for more separation before buttons
3. Container padding: Optionally increase `p-5` (20px) to `p-6` (24px) for overall breathing room

Recommended changes:
```tsx
// Headline: more space below
<p className="text-xl font-bold text-text-primary mb-4">

// Subtitle: slightly more space before buttons
<p className="text-base text-text-secondary mb-6">

// Container: slightly larger padding (optional but recommended)
<div className="p-6 flex-shrink-0">
```

**Verify**: Visual inspection - noticeable gaps between headline, subtitle, and buttons

**Done**: Clear visual separation exists between all header elements

---
